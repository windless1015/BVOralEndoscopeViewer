Download Accord libraries by Nuget console.

```shell
Install-Package Accord -Version 3.8.0
Install-Package Accord.Video.DirectShow -Version 3.8.0
Install-Package Accord.Controls.Imaging -Version 3.8.0
Install-Package Accord.Video -Version 3.8.0
Install-Package Accord.Controls -Version 3.8.0
Install-Package Accord.Video.FFMPEG -Version 3.8.0
Install-Package Accord.Vision -Version 3.8.0

```
